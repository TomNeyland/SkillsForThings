# Conductor

![Choose among seven controls for coordinated AI-assisted software builds: conducting for owned parallel work and integration; fan-and-critic for independent attempts and artifact review; autonomous-build for an explicitly invoked long build; and focused controls for feature depth, backlog ranking, session pacing, and commit history.](assets/conductor.png)

Conductor routes the decision currently at risk. Use `conducting` for a multi-agent team, `fan-and-critic` for an artifact that needs independent challenge, and `autonomous-build` to establish an explicitly requested long build. Inside that build, route the next decision to `autonomous-build-purpose-layers`, `autonomous-build-jealousy-ranking`, `autonomous-build-session-pacing`, or `autonomous-build-commit-essays`.
